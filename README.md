# Number Guessing Game (C++)

This is an interactive number guessing game built using C++.

## 🎯 Description
The program generates a random number between 1 and 100, and the user has to guess it. The program provides feedback whether the guess is too high or too low until the correct number is guessed.

## 🚀 Features
- Random number generation (1–100)
- User input validation
- Feedback system (Too High / Too Low)
- Attempt counter
- Performance evaluation (Excellent / Good / Practice)
- Replay functionality (play multiple times)

## 🛠️ Technologies Used
- C++

## ▶️ How to Run
1. Compile the program using any C++ compiler
2. Run the executable file
3. Enter your guesses as prompted
